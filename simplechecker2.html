Form Checks

The form below is controlled by Javascript using the following rules (configured automatically through the class attribute of the corresponding fields):

  • the C1 field must contain a number and be not empty
  • the C2 field must contain an email address

If the form fields are not valid, the user is warned and guided through an error message. These message style and position is fully customizable with CSS. The form cannot be submitted if its data is not valid.
